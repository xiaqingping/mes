# MES
使用 Vue.js + Ant Design vue 构建的后台管理系统

## 使用

``` bash
# 安装开发依赖
npm install

# 热重载开发服务器
npm run serve

# 生产
npm run build

# 运行测试
npm run test

# Lints and fixes files
npm run lint
```
